# Kubernetes Deployment for Food Delivery App\n\n## Quick Start (Docker Desktop K8s)\n\n1. Install kompose:\n   ```powershell\n   (New-Object Net.WebClient).DownloadFile('https://github.com/kubernetes/kompose/releases/download/v1.34.0/kompose-windows-amd64.exe', 'kompose.exe')\n   ```\n   Add to PATH: ` $env:Path += ';c:/Users/adali/Downloads/foodApp-main/foodApp-main' `\n\n2. Generate manifests:\n   `kompose convert --with-kompose-annotation=false -f docker-compose.yml -o k8s/`\n\n3. Review/edit generated files (postgres to StatefulSet, add secrets/resources).\n\n4. Apply:\n   `kubectl create ns fooddelivery`\n   `kubectl apply -f k8s/`\n\n5. Port forwards:\n   `kubectl port-forward -n fooddelivery deployment/frontend 5173:80`\n   `kubectl port-forward -n fooddelivery deployment/eureka-server 8761:8761`\n   `kubectl port-forward -n fooddelivery deployment/api-gateway 9000:9000`\n\nFrontend: http://localhost:5173\nEureka: http://localhost:8761\n\n## Manual Manifests (production-ready)\n- 01-namespace.yaml\n- 02-configmap-secrets.yaml\n- 03-postgres-statefulset.yaml\n- ...\n\nkubectl get all -n fooddelivery
